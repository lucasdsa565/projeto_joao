# projeto_joao

Este projeto consiste em uma página de apresentação pessoal (landing page) desenvolvida utilizando apenas HTML e CSS, com foco em design moderno, responsividade e boa experiência do usuário.

A interface destaca informações profissionais de forma clara e organizada, incluindo nome, cargo, descrição pessoal e links para redes sociais. O layout foi estruturado para ser visualmente atraente, com uso de tipografia adequada, espaçamento equilibrado e elementos gráficos que reforçam a identidade do desenvolvedor.

Entre os principais destaques do projeto estão:

Estrutura semântica em HTML;
Estilização moderna com CSS;
Layout responsivo e adaptável a diferentes telas;
Uso de ícones para redes sociais;
Design limpo e profissional.

O objetivo deste projeto foi praticar conceitos fundamentais de front-end, como organização de layout, hierarquia visual e criação de interfaces elegantes sem o uso de frameworks.
